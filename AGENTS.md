# 開発ガイドライン（メソロギア）

このディレクトリ配下の作業は本ガイドラインに従います。

## コメントと言語
- すべてのコメントは日本語で記述する
- セクション見出しコメントも日本語を用いる（英語見出しは避ける）

## 文字コード・改行・保存ルール
- 文字コード: UTF-8（BOM なし）
- 改行コード: LF
- 末尾改行: 常に付与

## 命名・スタイル
- CSS クラス名: 既存の命名に合わせて kebab-case を基本とする
- JS の変数・関数: camelCase を基本とする
- 定数: 必要に応じて SCREAMING_SNAKE_CASE
- UI 文言は日本語。変数名・ロジックは英語で統一（既存に合わせる）

## CSS の方針
- 主要カラーは可能な範囲で CSS カスタムプロパティ（var）に寄せる
- メディアクエリの基準幅は 1024px / 768px を優先的に使用
- フォーカス可視化は :focus-visible を用いる
- 動きの軽減（prefers-reduced-motion）に対応する
- 重複定義や順序依存を避け、できるだけ単一箇所に集約

## アクセシビリティ
- キーボード操作でのフォーカスリングを隠さない
- 可能であればコントラスト比を確保する（目標 AA）

## その他
- 既存のコードスタイルに合わせ、不要な大規模リファクタは避ける
- 変更は最小差分を心がけ、関連コメントも日本語で更新する
